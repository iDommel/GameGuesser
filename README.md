## Install & start:

Use the following

### (Optional) create a virtual environment
`python3 -m venv venv`

activate it:
on linux:
`source venv/bin/activate`
on windows:
`./venv/Scripts/activate.ps1`
### Install dependencies:
`pip install .`

### Run the server:
`.main.py`

## Front Part

### Install dependencies:

in `gameguesserfront` folder
`npm install`

### Run the front:
`npm start`
